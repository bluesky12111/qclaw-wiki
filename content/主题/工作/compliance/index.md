# 泰诺麦博合规制度修订建议评估报告
## 结合2026年5月1日新规的深度分析

**生成日期**：2026年4月18日  
**分析依据**：最高人民法院、最高人民检察院《关于办理贪污贿赂刑事案件适用法律若干问题的解释（二）》（2026年5月1日施行）

---

## 一、外部政策环境总览

### 1.1 三份核心文件的叠加效应

| 文件 | 核心变化 | 对新替妥的影响 |
|:---|:---|:---|
| **《解释（二）》** | 医疗领域行贿从严从重；10万/20万入刑标准；企业无法"甩锅"给代表 | 🔴 **生死红线** |
| **药品价格机制改革** | 首发价格市场化但需临床价值支撑；医保支付是终极定价机制 | 🟢 **利好创新药** |
| **行业协会商会新政** | 会议/培训/课题全部可追溯；协会不再是"安全通道" | 🟡 **渠道重构** |

### 1.2 核心判断：医药行业底层规则重构

```
旧模式：高定价 → 预留回扣空间 → 带金销售 → 关系驱动
新模式：临床价值 → 合理定价 → 学术推广 → 合规驱动
```

**新替妥的定位优势**：作为创新单抗，具备"强FABE专业属性和学术力"，天然适配新规则。

### 【不予修订部分】

#### 1. 无劳务付费小型会照片要求

| 评估维度 | 分析 |
|:---|:---|
| **建议内容** | 照片无需显示会议时间，≥1张即可 |
| **政策符合性** | ❌ **不符合合规趋势** |
| **合规风险** | 🔴 **高** |
| **具体评估** | 《解释（二）》背景下，**可追溯性**是合规的生命线。照片是证明"真实发生"的核心证据。时间戳可防止"一套照片多次使用"，照片数量可证明参会人数。放松这一要求，等于给企业留下"伪学术"操作空间，一旦被查，百口莫辩。**合规部"不予修订"的决定是正确的。** |
| **建议** | ✅ **维持现状，且应强化执行** |
