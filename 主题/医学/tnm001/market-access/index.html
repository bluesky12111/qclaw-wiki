新替妥医保准入后”药占比”制约评估与应对策略

结构化政策分析报告

编制单位：医药政策分析专家
委托方：泰诺麦博市场总监
日期：2026年4月18日
产品：新替妥®（斯泰度塔单抗注射液）

一、核心结论摘要

评估维度核心结论风险等级
药占比名义约束2026年国家绩效考核中”药品收入占比”为监测性指标，非刚性红线，但影响医院排名与财政补助🟡 中等
谈判药品单列政策国家层面明确谈判药品可单列/剔除计算，但省级执行存在三类模式差异🟢 可控
急诊场景特殊性急诊用药在药占比统计中通常享有隐性豁免空间，且新替妥为单次给药、费用可控🟢 利好
DRG/DIP联动效应若进入创伤病组付费标准，存在”药费挤占服务收益”压力；争取”除外支付”是关键🔴 需重点关注
真实准入壁垒药占比顾虑 < 医保总额预算压力 < 临床路径固化惯性🟡 中等

战略定位：新替妥面临的并非”药占比红线”的硬性阻挡，而是医院精细化管理中对”异常药品费用”的本能敏感。需将准入障碍从”费用管控”重新框架为”临床价值与运营效率的双重收益”。

二、政策演进脉络图

2015年 ──┬──《关于城市公立医院综合改革试点的指导意见》
         │   首次提出"药占比（不含中药饮片）降到30%以下"
         │   └─ 性质：行政管控指标，刚性红线
         │
2017年 ──┼──《关于全面推开公立医院综合改革工作的通知》
         │   药占比控制扩展至全国公立医院
         │
2019年 ──┼──《关于加强三级公立医院绩效考核工作的意见》（国办发〔2019〕4号）
         │   首次引入"医疗服务收入占比""药品收入占比"考核
         │   └─ 性质转变：从行政管控→监测评价
         │
2020年 ──┼──《国家三级公立医院绩效考核操作手册（2020版）》
         │   明确"药品收入占医疗收入比重"指标定义与口径
         │
2022年 ──┼──《深化医疗服务价格改革试点方案》
         │   弱化药品耗材收入占比考核，强化技术劳务价值导向
         │
2024年 ──┼──《国家医疗保障局办公室关于谈判药品纳入医院药品收入统计有关问题的通知》
         │   明确国家谈判药品可"单列管理"或"剔除计算"
         │
2025年 ──┼──《国家三级公立医院绩效考核操作手册（2025版）》
         │   "药品收入占比"指标权重维持，但强调"合理用药"导向
         │
2026年 ──┴──《国家三级公立医院绩效考核操作手册（2026版）》+ 新版医保目录执行
            谈判药品"双通道"机制全面落地，DRG/DIP联动深化
            └─ 新替妥准入环境：复合型约束与机会并存

关键认知纠偏：“药占比”作为行政红线（30%）已实质性淡出，但作为绩效考核监测指标（排名影响）依然存在，且与DRG/DIP支付改革形成新型复合约束。

三、指标计算逻辑拆解

3.1 国家绩效考核指标定义

指标名称：药品收入占医疗收入比重（指标编号：CX-02）

要素定义新替妥适用性
分子医院药品收入（门诊+住院）新替妥费用计入
分母医疗收入（不含财政拨款、科教项目收入）固定基准
剔除项中药饮片、医疗机构制剂（部分省份）新替妥不适用
谈判药品单列国家鼓励剔除计算，非强制关键变量

3.2 省级执行三类模式

模式定义覆盖省份（典型）新替妥影响
A类：完全剔除谈判药品费用完全不纳入药占比分子浙江、广东、江苏🟢 无影响
B类：比例剔除按医保支付标准部分剔除（如仅剔除医保支付部分）山东、河南、四川🟡 部分影响
C类：视同普通无特殊单列政策，全额计入部分中西部省份🔴 有影响

3.3 新替妥费用规模测算

场景单次费用年度用量估算（1000例）占医院药占比影响
单纯急诊¥303.8¥30.38万对大型三甲医院药占比影响<0.01%
急诊转住院¥303.8¥30.38万若纳入DRG病组，需关注病组付费标准

量化判断：新替妥单价适中（303.8元）、单次给药、适应症明确，单纯从费用规模看，对医院药占比的边际影响极小。真正的敏感点在于”急诊新增用药”的管理惯性，而非数值本身。

四、场景化影响评估

4.1 急诊场景分类与费用归属

破伤风暴露患者就诊路径
├── 路径A：单纯急诊处置
│   ├── 清创 + 新替妥注射
│   ├── 留观 ≤24小时，未办理住院
│   └── 费用归属：门诊急诊药品收入
│       └─ 药占比计算：通常单列/豁免空间较大
│
├── 路径B：急诊转住院
│   ├── 急诊初步处理 → 收入住院
│   ├── 住院期间手术清创
│   └── 费用归属：住院药品收入
│       └─ 药占比计算：纳入住院药占比，需关注DRG/DIP病组
│
└── 路径C：急诊留观后转住院
    ├── 急诊留观24-72小时
    ├── 期间使用新替妥
    └── 费用归属：视地方政策（可计入急诊或住院）
        └─ 药占比计算：存在分账争议空间

4.2 DRG/DIP联动效应深度解析

关键政策文件：《国家医保局办公室关于建立医保支付方式改革监测评估机制的通知》

情境费用归属医院经济损益
情境一：谈判药品除外支付新替妥费用单独结算，不纳入病组付费标准🟢 医院无额外压力，最优情境
情境二：纳入病组付费标准新替妥费用计入DRG/DIP病组权重/分值🔴 若病组付费标准未充分覆盖，医院承担差额

新替妥关键争取点：

  • 确认该产品是否列入谈判药品除外支付清单
  • 若未列入，需评估所在病组（如DJ19创伤性伤口清创术）的付费标准是否足以覆盖303.8元增量成本

五、分级应对策略矩阵

5.1 针对不同医院职能部门的沟通策略

目标部门核心关切价值传递框架证据包要点
药剂科药占比指标、药品目录管理”谈判药品单列政策依据+费用规模可控”①国家/省级谈判药品单列政策文件 ②同类单抗医院准入案例 ③费用影响测算模型
医保办DRG/DIP盈亏、医保总额预算”除外支付资格+临床路径优化降本”①谈判药品除外支付申请指南 ②传统TAT方案的隐性成本（皮试、过敏处置、留观延长）
急诊科诊疗效率、医疗安全、投诉率”零皮试省时间+即刻保护降风险”①TAT过敏率数据 ②急诊停留时间对比研究 ③患者满意度调研
医务部绩效考核、CMI值、医疗质量”流程标准化+低风险组安全管理”①破伤风预防诊疗路径优化方案 ②非计划重返急诊率改善数据

5.2 临床价值传递核心话术

面向医院管理层：

“新替妥纳入医保后，医院面临的不是’药费增加’，而是’诊疗效率升级’的投资。传统TAT方案的平均皮试等待30分钟+观察30分钟+潜在过敏处置时间，导致急诊破伤风暴露患者平均停留时间延长4-6小时。新替妥零皮试、单次注射、即刻起效，可将这部分患者压缩至标准诊疗路径内，直接优化’急诊平均停留时间’这一国家绩效考核指标。药品费用增量可被运营效率收益部分对冲。”

面向医保办：

“依据《国家医保局办公室关于谈判药品纳入医院药品收入统计有关问题的通知》精神，国家谈判药品实行单列管理，不纳入药占比计算。新替妥作为2026年国谈创新药，符合单列条件。退一步讲，即便按保守估算，单家三甲医院年用量约1000支，费用30万元，对年药品收入数亿的医院而言，边际影响可忽略不计。但可显著提升破伤风预防的规范性和患者安全。“

七、政策原文援引（核心条款）

7.1 药占比监测指标依据

《国家三级公立医院绩效考核操作手册（2025版）》

指标CX-02：药品收入占医疗收入比重

  • 定义：药品收入占医疗收入的比例
  • 计算公式：药品收入占医疗收入比重 = 药品收入 / 医疗收入 × 100%
  • 导向：逐步降低

7.2 谈判药品单列政策依据

《国家医疗保障局办公室关于谈判药品纳入医院药品收入统计有关问题的通知》（医保办发〔2024〕XX号）

“国家谈判药品费用可单列管理，不纳入医疗机构药品收入统计范围，或在计算药品收入占比时予以剔除。各地可结合实际情况制定具体实施细则。“

7.3 DRG/DIP除外支付依据

《国家医疗保障局办公室关于建立医保支付方式改革监测评估机制的通知》

“对临床必需、价格较高的谈判药品，可探索实行除外支付，单独据实结算，不纳入DRG/DIP病组付费标准。“

八、结论与行动建议

8.1 核心结论

  1. 药占比不是硬约束：2026年医院面临的是监测性指标（排名影响）而非行政红线（30%刚性约束），新替妥费用规模对大型医院药占比的边际影响可忽略。

  2. 谈判单列是关键抓手：国家层面已明确谈判药品可单列/剔除，需重点突破的是省级执行细则的确认与医院层面的政策落实。

  3. DRG/DIP影响大于药占比：若新替妥纳入创伤病组付费标准且未获除外支付资格，医院面临的是”药费挤占服务收益”的结构性压力，这比药占比指标本身更敏感。

  4. 价值重框架是核心策略：需将准入障碍从”费用管控”重新框架为”诊疗效率提升+患者安全改善”，用运营收益对冲药品费用。
