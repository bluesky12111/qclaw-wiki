泰诺麦博区域市场部（RPM）绩效评价标准方案（2026年版）

制定日期：2026年4月15日
适用对象：区域推广经理（RPM）
评价周期：季度评价 + 年度总评
核心原则：结果导向60% + 关键过程30% + 行为兜底10%

一、权重架构总览

指标模块权重结果/过程属性核心理由四新标准对应
专家网络15%结果导向讲者力（角色力）是四新标准核心产出，讲者质量决定学术影响力角色力
项目管理25%结果导向科室力（活动后科室会转化）是项目管理的最终价值锚点科室力
宣传物料20%结果导向品牌力+科普力（五位一体落地）是物料建设的终极目标品牌力+科普力
市场分析10%关键过程为结果提供方向，洞察深度决定策略有效性—
平台搭建10%关键过程平台是手段，搭建质量影响后续产出—
培训赋能10%关键过程能力建设是过程，赋能效果体现在后续结果指标中—
行为指标10%文化兜底价值观与职业素养，一票否决项—
合计100%结果60%:过程30%:行为10%——

二、指标详细定义

【结果导向指标】价值产出（60%）

1. 专家网络（15%）— 讲者力（角色力）

指标定义：通过学术活动挖掘、培养、评估讲者的质量，建立可持续输出的新替妥专家梯队。

评价维度权重评价标准评分细则
讲者质量（内部评估）10%季度讲者授课内部评估平均分≥80分100分：≥90分；85分：80-89分；70分：70-79分；<70分：<70分
讲者活跃5%季度讲者授课场次≥N场100分：达标；80分：80%达标；60分：60%达标；<60分：不达标

讲者质量评估维度（公司内部评估）：

  • 内容准确性（30%）：产品知识、临床数据传递是否准确
  • 表达生动性（30%）：授课感染力、听众互动情况
  • 产品理解度（40%）：对新替妥差异化优势的把握深度

数据来源：学术活动记录表、讲者评估表、科室会反馈表

2. 项目管理（25%）— 科室力（核心结果）

指标定义：学术会议后的跟进转化效果，衡量活动对科室的实际渗透力（科室力）。

评价维度权重评价标准评分细则
科室会转化15%大型会议后30天内组织跟进科室会≥1场100分：100%转化；80分：80%转化；60分：60%转化；<60分：<60%
科室覆盖深度10%季度覆盖核心科室数量≥N个，且科室会质量达标100分：达标且质量优秀；80分：达标；60分：80%达标；<60分：不达标

关键动作：

  • 会前与讲者沟通，明确科室会转化意向
  • 会后3天内与参会HCP回访，趁热打铁推动科室会
  • 科室会执行质量（参与人数、时长、反馈评分）

数据来源：项目管理台账、科室会记录表、HCP回访记录

3. 宣传物料（20%）— 品牌力+科普力+五位一体

指标定义：科普作品产出质量、品牌内容二次传播效果，以及科室场景标准化建设（五位一体）的落地。

评价维度权重评价标准评分细则
科普作品产出10%季度产出可传播科普作品≥N个100分：达标且质量优秀；80分：达标；60分：80%达标；<60分：不达标
品牌传播覆盖5%内容二次传播覆盖人次≥N100分：达标；80分：80%达标；60分：60%达标；<60分：不达标
五位一体落地5%重点医院完成科室场景标准化建设≥N家100分：达标；80分：80%达标；60分：60%达标；<60分：不达标

科普作品标准：

  • 形式：短视频、图文、海报、手册等
  • 质量：医学准确性（医学部审核）、传播性（市场部评估）
  • 应用：可在科室、公众号、患教场景使用

五位一体科室场景：

  1. 大厅科普视频/海报
  2. 走廊破伤风规范诊疗路径海报
  3. 护理台科普手册/易拉宝
  4. 诊疗台旋转台卡
  5. 医生知情告知书

数据来源：科普作品清单、传播数据后台、科室场景验收表

【关键过程指标】能力建设（30%）

4. 市场分析（10%）— 策略方向

指标定义：对区域市场的洞察深度与策略规划质量，为结果指标提供方向指引。

评价维度权重评价标准评分细则
分析质量5%季度市场分析报告质量100分：洞察深刻、数据详实、策略可行；80分：结构完整、分析到位；60分：基本合格；<60分：质量不达标
策略匹配5%策略规划与实际执行匹配度100分：策略精准、执行顺畅；80分：策略可行、偶有偏差；60分：策略基本可行；<60分：策略与执行脱节

关键产出：

  • 区域竞争格局分析（TAT/HTIG/竞品份额）
  • 重点医院潜力评估与分级
  • 季度策略规划（准入/活动/讲者/物料的优先级排序）

数据来源：市场分析报告、策略规划文档、执行复盘记录

5. 平台搭建（10%）— 基础建设

指标定义：学术平台与合作网络的规划与建设质量，是后续学术活动的基础。

评价维度权重评价标准评分细则
平台覆盖5%区域学术平台（学会/协会/医院联盟）覆盖100分：覆盖全部核心平台；80分：覆盖80%；60分：覆盖60%；<60分：<60%
项目设计5%合作项目设计质量100分：设计优秀、资源到位；80分：设计可行、基本落实；60分：设计一般；<60分：设计粗糙

关键产出：

  • 区域急诊/外科/创伤学术平台合作清单
  • 年度合作项目规划（区域会、病例大赛、共识巡讲等）
  • 平台资源整合（专家资源、媒体资源、渠道资源）

数据来源：平台合作清单、项目规划文档、合作协议

6. 培训赋能（10%）— 能力建设

指标定义：对MR团队及合作方的培训赋能质量，提升整体推广能力。

评价维度权重评价标准评分细则
培训执行5%季度组织培训≥N场100分：达标且覆盖全面；80分：达标；60分：80%达标；<60分：不达标
知识库贡献5%FAQ更新、案例库建设、课件开发100分：贡献突出、质量优秀；80分：有贡献、质量良好；60分：有贡献；<60分：贡献不足

培训内容：

  • 新替妥产品知识更新
  • 竞品对比与应对话术
  • 政策环境解读（医保/准入）
  • 科室会演讲技巧

关键产出：

  • 培训记录与签到表
  • 培训课件（PPT/视频）
  • FAQ更新文档
  • 优秀案例库

数据来源：培训记录、课件清单、知识库更新日志

三、行为指标（10%）— 文化兜底

评价维度（保持原5个维度，各2%）

维度评价要点红线行为（一票否决）
团队协作与MR/大区/市场部的配合度，信息共享及时性故意隐瞒信息、恶意竞争、破坏团队氛围
主动创新主动发现问题、提出改进建议、尝试新方法因循守旧、拒绝改变、消极执行
执行力任务完成及时性、质量标准、闭环管理拖延怠工、质量不达标、有始无终
学习成长主动学习产品/竞品/政策知识，自我提升拒绝学习、知识老化、能力停滞
合规意识严格遵守公司合规制度、学术推广规范学术造假、利益输送、违反合规红线

评分规则：

  • 100分：表现卓越，可作为标杆
  • 80分：表现良好，符合预期
  • 60分：基本合格，需改进
  • <60分：不合格，需重点关注

一票否决：出现红线行为，该季度绩效直接降档处理（优秀→良好，良好→合格，合格→不合格）。

四、评分标准与系数

等级系数表

等级系数区间评价标准
卓越120%≥115分超越预期，可推广复制
优秀100%95-114分达成预期，表现稳定
较好85%80-94分基本达成，个别亮点
合格70%60-79分勉强达标，需改进
不合格50%<60分未达标，需重点关注

总分计算公式

总分 = Σ(指标得分 × 指标权重) × 行为系数

其中：
- 结果指标：专家网络15% + 项目管理25% + 宣传物料20% = 60%
- 过程指标：市场分析10% + 平台搭建10% + 培训赋能10% = 30%
- 行为指标：10%
- 合计：100%（封顶110分）

示例计算

某RPM季度评价：

  • 专家网络：85分 × 15% = 12.75
  • 项目管理：90分 × 25% = 22.50
  • 宣传物料：80分 × 20% = 16.00
  • 市场分析：90分 × 10% = 9.00
  • 平台搭建：80分 × 10% = 8.00
  • 培训赋能：85分 × 10% = 8.50
  • 行为指标：90分 × 10% = 9.00

加权总分：12.75+22.50+16.00+9.00+8.00+8.50+9.00 = 85.75分

等级：较好（85%系数）

五、评委权重

评委权重评价重点
赵总（营销中心总经理）40%战略一致性、最终结果、跨部门协同
雷磊（市场部负责人）30%策略质量、活动质量、物料质量、讲者质量
分管大区经理30%执行配合度、团队反馈、区域适应性

评价流程：

  1. RPM提交季度自评报告（含数据佐证）
  2. 评委分别打分，系统自动加权计算
  3. 三方合议（如有分歧，以赵总意见为准）
  4. 反馈面谈，制定下季度改进计划

六、关键术语对照表

本方案术语四新标准对应五位一体对应说明
专家网络（结果）角色力—讲者挖掘、培养、评估
项目管理（结果）科室力—活动后科室会转化
宣传物料（结果）品牌力+科普力五位一体科普产出+科室场景建设
市场分析（过程）——洞察与策略方向
平台搭建（过程）——学术平台建设
培训赋能（过程）——能力建设

七、附则

  1. 解释权：本方案由泰诺麦博营销中心市场部负责解释。
  2. 调整机制：每半年回顾一次，根据市场变化和执行反馈调整权重与标准。
  3. 特殊情况：如遇重大政策变化（如医保目录调整、竞品上市），可临时调整指标权重。
  4. 生效日期：2026年Q2起执行，原2024年Q4版本同时废止。
